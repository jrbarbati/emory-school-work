Running
python main.py <data file> <k> <output filename> <class column number>

Example command:
python main.py dataset1.csv 10 res.txt 6
	i.e. dataset1.csv has a class column which is column 6 (not starting at 0)

python main.py dataset2.csv 10 res2.txt 1
	i.e. dataset2.csv has a class column which is column 1

*** NOTE ***
Only supports CSV files
